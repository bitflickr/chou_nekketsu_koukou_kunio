## 2. 游戏模式

### 2.1 世界杯模式（单人）

- 玩家操控**日本队**，依次挑战其他 7 个国家队
- 对战顺序固定：印度 → 冰岛 → 中国 → 非洲（肯尼亚）→ 英国 → 美国 → 苏联
- 每场比赛独立，无连续体力消耗
- 全部通关后显示通关画面

### 2.2 VS 对战模式（本地双人）

- 两名玩家各选一支队伍
- 单场淘汰制
- 支持选择全部 8 支队伍

### 2.3 Bean Ball 模式

- 变体规则：取消外场，所有 6 人在内场
- 被击中的球员直接淘汰（不转外场）
- 更快节奏的淘汰赛

## 3. 场地规则

### 3.1 球场布局

```
┌─────────────────────────────────────────────┐
│              外场（B队外场球员）               │
│  ┌───────────────────┬───────────────────┐  │
│  │                   │                   │  │
│  │   A队内场（3人）   │   B队内场（3人）   │  │
│  │                   │                   │  │
│  │                   │                   │  │
│  └───────────────────┴───────────────────┘  │
│              外场（A队外场球员）               │
└─────────────────────────────────────────────┘
```

- 球场分为**左半场**和**右半场**，中线分隔
- 每队 3 名内场球员站在己方半场
- 每队 3 名外场球员分布在对方半场的**外围三侧**（上、下、对面）
- 内场球员不可越过中线
- 外场球员不可进入内场区域

### 3.2 场地尺寸（逻辑坐标）

| 参数 | 值 |
|------|------|
| 球场总宽度 | 256 px |
| 球场总高度 | 192 px |
| 内场宽度（单侧） | 96 px |
| 内场高度 | 128 px |
| 外场通道宽度 | 32 px |

### 3.3 区域约束

- 内场球员：限制在己方内场矩形内移动
- 外场球员：限制在对方半场外围通道内移动
- 持球球员可在约束区域内自由移动
- 无球球员在约束区域内自动跟随球的位置进行基础站位

## 4. 角色系统

### 4.1 属性

每个角色拥有以下基础属性：

| 属性 | 缩写 | 说明 | 范围 |
|------|------|------|------|
| 体力 | HP | 被击中后扣减，归零则淘汰 | 1~64 |
| 攻击力 | ATK | 影响投球伤害 | 1~16 |
| 防御力 | DEF | 减少受到的伤害 | 1~16 |
| 速度 | SPD | 移动速度 | 1~16 |
| 跳跃力 | JMP | 跳跃高度和滞空时间 | 1~16 |

### 4.2 角色状态

角色在比赛中处于以下状态之一：

| 状态 | 说明 |
|------|------|
| **空闲 (Idle)** | 站立不动 |
| **移动 (Move)** | 8 方向移动 |
| **跳跃 (Jump)** | 起跳、滞空、落地 |
| **持球 (Hold)** | 手持球，可移动、投球、传球 |
| **投球 (Throw)** | 投球动画，短暂不可移动 |
| **接球 (Catch)** | 接球判定窗口中 |
| **被击中 (Hit)** | 受到伤害，击退动画 |
| **倒地 (Down)** | 被击倒后短暂倒地，有无敌帧 |
| **淘汰 (Out)** | HP 归零，移至外场（或退出比赛） |

### 4.3 角色控制

**玩家控制（触屏）：**
- 虚拟摇杆：8 方向移动
- A 按钮：投球 / 传球（持球时）；接球（无球且球接近时）
- B 按钮：跳跃（空中投球 = 跳投）
- C 按钮：下蹲 / 闪避（按住生效，可躲直线球；键盘 Shift/C）
- 投球方向：摇杆方向决定投球目标方向

**AI 控制：**
- 详见第 9 节 AI 系统

## 5. 球的物理系统

### 5.1 球的状态

| 状态 | 说明 |
|------|------|
| **被持有 (Held)** | 跟随持球角色 |
| **飞行 (Flying)** | 投球后空中飞行 |
| **滚动 (Rolling)** | 落地后在地面滚动 |
| **静止 (Idle)** | 停止不动，可被拾取 |

### 5.2 投球物理

投球初速度 = 基础速度 × (1 + ATK / 16)；球速随飞行距离衰减：`speed = initial_speed × (1 - distance / max_range × 0.5)`；低速球更容易被接住。

**投球类型（轨迹，依据原版 Super Dodge Ball，非全部抛物线）：**

| 类型 | 触发 | 轨迹 | 可否下蹲躲 |
|------|------|------|-----------|
| **直线球 (Straight)** | 地面持球投 | 贴身高度（约 10px）**水平直线快球**，不受重力；撞墙后才下落→反弹→滚动 | ✅ 可（从头顶飞过） |
| **跳投 (Jump)** | 跳跃中持球投 | 从跳跃高度**斜向下坠**飞向目标 | ❌ 不可（来球高于下蹲） |
| **吊球/传球 (Lob)** | 传球；高抛类必杀 | 大抛物线（上抛后落下） | ✅ 可（视高度） |

> 修订说明：旧版本将所有投球都建模为抛物线；经对照原版（StrategyWiki NES Gameplay、Famicom 官方说明书）已修正为上述三类。普通地面投球应为直线快球。

### 5.3 必杀技投球（详见第 6 节）

- 必杀技球有特殊轨迹，不遵循普通抛物线
- 必杀技球伤害更高
- 必杀技球更难被接住

### 5.4 传球

- 持球球员可向己方队友传球
- 传球速度较慢，无伤害
- 内场 ↔ 外场之间可传球
- 传球用于战术配合和转移进攻方向

### 5.5 反弹

- 球击中地面后反弹，高度逐次降低
- 反弹 3 次后变为滚动状态
- 滚动球速度逐渐衰减至 0 → 静止

### 5.6 拾球

- 任何球员靠近静止/滚动的球可自动拾取
- 拾取后球变为被持有状态

## 6. 必杀技系统

### 6.1 触发条件

必杀技投球在以下条件下触发：
1. 球员**持球**状态
2. 进行**助跑投球**（移动中按投球键）或**跳跃投球**（空中按投球键）
3. 部分角色需要特定条件（如 HP 低于一定值）

### 6.2 必杀技分类

| 类型 | 轨迹描述 | 代表角色 |
|------|----------|----------|
| **旋转球** | 球高速旋转，轨迹螺旋弯曲 | くにお（国尾） |
| **闪电球** | 球带电光特效，直线高速 | 部分美国队角色 |
| **分裂球** | 球在接近目标时分裂成多个分身 | 部分印度队角色 |
| **变速球** | 球先慢后快，突然加速 | 部分冰岛队角色 |
| **消失球** | 球中途短暂消失再出现 | 部分中国队角色 |
| **炸弹球** | 球击中后爆炸范围伤害 | 部分苏联队角色 |
| **蛇形球** | 球走 S 形轨迹 | 部分非洲队角色 |
| **高抛球** | 球高高抛起后垂直砸下 | 部分英国队角色 |

### 6.3 必杀技伤害

- 必杀技伤害 = 普通投球伤害 × 必杀技倍率（1.5~3.0）
- 具体倍率见 `original_game_data.md`

### 6.4 必杀技防御

- 必杀技球可以被接住，但判定窗口更小
- 接必杀技球失败会受到伤害并击退更远
- 闪避（跳跃躲开）是最可靠的防御方式

## 7. 接球与闪避系统

### 7.1 接球

- 当球飞向角色时，角色可按接球键尝试接球
- 接球判定窗口：球到达角色碰撞框前的 **6 帧**（约 100ms @ 60fps）
- 接球成功条件：
  - 角色面朝球的方向
  - 在判定窗口内按下接球键
  - 球速未超过角色 DEF 对应的可接球速度阈值
- 接球失败：球击中角色，造成伤害

### 7.2 接球成功效果

- 球变为被持有状态
- 角色获得短暂无敌帧（防止连续攻击）
- 可立即反击投球

### 7.3 闪避

- **下蹲（C 键，按住生效）可躲过直线球**：球从头顶飞过，不造成伤害
- **跳投系不可被下蹲躲过**（来球从上方斜降）
- 跳跃（B 键）中角色有较小的碰撞框，亦可用于躲避部分来球
- 闪避不消耗任何资源，但有动作恢复时间

### 7.4 被击中

- 球击中角色后：
  - 扣减 HP：`damage = ball_speed × ATK_thrower / DEF_target`
  - 角色进入**被击中**状态，播放击退动画
  - 击退距离与球速成正比
  - 被击中后有短暂无敌帧（约 30 帧）
- HP 归零：
  - 内场球员 → 转移至外场（淘汰出内场）
  - 如果是 Bean Ball 模式 → 直接退出比赛

## 8. 比赛流程

### 8.1 开球

1. 比赛开始时，球在中场
2. 跳球判定：双方各一名角色跳起争球
3. 跳球胜者获得首次持球权

### 8.2 回合流程

```
持球方投球 → 球飞行 → 命中/接住/落地
  ├── 命中：目标受伤害，球反弹后可被任意方拾取
  ├── 接住：接球方获得持球权
  └── 落地：球滚动/静止，可被任意方拾取
```

### 8.3 内外场切换

- 内场球员 HP 归零 → 移至外场
- 外场球员击中对方内场球员使其 HP 归零 → 该外场球员可返回内场（原版规则）
- 返回内场的球员 HP 不恢复

### 8.4 胜负判定

- **一方内场球员全部被淘汰** → 对方胜利
- 无时间限制（原版无计时）

### 8.5 比赛结束

- 胜利方播放庆祝动画
- 世界杯模式：进入下一场对战
- VS 模式：返回选队画面

## 9. AI 系统

### 9.1 AI 状态

AI 控制的角色在以下行为间切换：

| 行为 | 触发条件 | 说明 |
|------|----------|------|
| **巡逻** | 无球且远离球 | 在区域内随机移动，保持基础站位 |
| **追球** | 球在可拾取范围内 | 移向球的位置 |
| **投球** | 持球中 | 选择目标 → 选择投球方式 → 投球 |
| **传球** | 持球中且不适合投球 | 传给更好位置的队友 |
| **接球** | 球飞向自己 | 判断是否尝试接球 |
| **闪避** | 球飞向自己且速度过高 | 跳跃闪避 |
| **站位** | 无球且球在对方控制 | 移到防守/进攻有利位置 |

### 9.2 难度等级

| 等级 | 对应队伍 | AI 特征 |
|------|----------|---------|
| Easy | 印度、冰岛 | 接球反应慢、投球方向随机、少用必杀技 |
| Normal | 中国、非洲 | 正常反应速度、有目标选择、偶尔使用必杀技 |
| Hard | 英国、美国 | 快速反应、选择弱目标、频繁使用必杀技 |
| Expert | 苏联 | 极快反应、完美站位、每次投球都用必杀技 |

### 9.3 目标选择

AI 投球目标选择优先级：
1. HP 最低的内场球员
2. 距离最近的球员
3. 正在恢复中（被击倒刚起身）的球员

## 10. 触屏适配（移动端）

### 10.1 虚拟摇杆

- 左侧屏幕区域：虚拟摇杆
- 触摸起点为摇杆中心
- 支持 8 方向输入
- 摇杆半径：48px（可配置）
- 释放后自动回中

### 10.2 动作按钮

- 右侧屏幕区域：A / B 两个按钮
- A 按钮（右下）：投球 / 传球 / 接球
- B 按钮（右上）：跳跃 / 闪避
- 按钮大小：64×64 px（可配置）
- 支持按钮半透明度调节

### 10.3 屏幕适配

- 基础分辨率 256×240（NES 原版）
- 等比缩放至设备分辨率
- 横屏固定
- 安全区域适配（刘海屏、圆角）
- UI 元素不遮挡核心游戏区域

## 11. 视觉与音频

### 11.1 像素美术

- 角色精灵：16×24 px（NES 原版尺寸）
- 球精灵：8×8 px
- 球场贴图：256×240 px 全屏背景
- 动画帧数：
  - 站立：2 帧
  - 行走：4 帧
  - 跳跃：3 帧（起跳、滞空、落地）
  - 投球：3 帧
  - 被击中：2 帧
  - 倒地：1 帧
  - 必杀技：4~6 帧（角色动作 + 球特效）

### 11.2 调色板

- 使用 NES 调色板（54 色）作为基准
- 每个队伍有独特的配色方案
- 球场根据对战地点切换背景

### 11.3 音频需求

| 类别 | 内容 |
|------|------|
| BGM | 主菜单曲、比赛曲（至少 2 首）、胜利曲、失败曲、通关曲 |
| SE - 球 | 投球、接球、击中、反弹、必杀技发动 |
| SE - 角色 | 跳跃、被击中叫声、淘汰、欢呼 |
| SE - 系统 | 菜单选择、比赛开始哨声、比赛结束哨声 |

### 11.4 画面特效

- 必杀技球：拖尾粒子效果（闪电、火焰、旋风等）
- 被击中：闪烁 + 击退
- 淘汰：角色闪烁后消失
- 场地切换：简单淡入淡出
