- **SPD**: 速度（移动速度），范围 1~16
- **JMP**: 跳跃力（跳跃高度/滞空），范围 1~16

### 3.3 必杀技轨迹详细描述

**旋风球（くにお）**
- 球从投出位置开始螺旋旋转
- 旋转半径从 0 逐渐增大到 16px，再收缩回 0
- 整体沿直线方向前进
- 旋转速度 720°/秒

**分裂球（チャパティ）**
- 球飞行至距目标 48px 时分裂
- 产生 2 个幻影球（无伤害），1 个真球
- 3 个球呈扇形分散
- 幻影球碰到目标后消失

**消失球（チンリン）**
- 球飞行至 1/3 距离时消失（alpha → 0）
- 消失持续约 20 帧
- 在距目标 32px 处重新出现
- 消失期间仍有碰撞判定

**炸弹球（ボリス）**
- 球高速直线飞行
- 击中目标后产生爆炸效果
- 爆炸半径 32px
- 范围内其他角色受到 50% 伤害

## 4. 伤害公式

### 4.1 基础伤害

```
base_damage = ball_speed × (ATK_thrower / 8)
actual_damage = base_damage × (8 / DEF_target)
```

简化：
```
damage = ball_speed × ATK_thrower / DEF_target
```

### 4.2 必杀技伤害

```
special_damage = damage × special_multiplier
```

### 4.3 球速计算

```
initial_ball_speed = BASE_SPEED + (ATK_thrower × SPEED_PER_ATK)

其中：
  BASE_SPEED = 3.0（基础像素/帧）
  SPEED_PER_ATK = 0.25（每点攻击力增加的速度）
```

### 4.4 接球速度阈值

```
catch_threshold = BASE_CATCH + (DEF_target × CATCH_PER_DEF)

其中：
  BASE_CATCH = 4.0
  CATCH_PER_DEF = 0.5

如果 ball_speed > catch_threshold：
  接球判定窗口减半
如果 ball_speed > catch_threshold × 1.5：
  无法接球（只能闪避）
```
