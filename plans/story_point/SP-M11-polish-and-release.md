## 技术要点

- **测试清单**（关联 RISK-09）：
  - [ ] 世界杯模式 7 场全通
  - [ ] VS 模式所有队伍互相对战（8×7=56 组合抽测）
  - [ ] Bean Ball 模式完整流程
  - [ ] 17 种必杀技全部触发验证
  - [ ] 存档/读档功能验证
  - [ ] 中断恢复（来电、锁屏、切后台）
  - [ ] 各分辨率/设备验证
  - [ ] 内存泄漏检测（长时间运行）

- **数值平衡目标**：
  | 指标 | 目标范围 |
  |------|---------|
  | 普通球击杀内场（Easy队） | 5~8 次击中 |
  | 普通球击杀内场（Expert队） | 3~5 次击中 |
  | 必杀技击杀内场 | 1~3 次击中 |
  | 单场比赛时长 | 1~3 分钟 |
  | 世界杯全通时长 | 15~25 分钟 |

- **应用商店要求**：
  | 平台 | 要求 |
  |------|------|
  | Google Play | APK/AAB < 150MB，API 33+，隐私政策 |
  | App Store | 截图（6.5"/5.5"），年龄分级，隐私标签 |
