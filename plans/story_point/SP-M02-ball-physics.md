## 技术要点

- **球速公式**（来自 original_game_data.md）：
  ```
  initial_ball_speed = 3.0 + (ATK × 0.25)  # 像素/帧
  ```

- **球状态机**：
  ```
  Held → Flying（投球/传球触发）
  Flying → Rolling（落地反弹 3 次后）
  Flying → Held（被接住）
  Rolling → Idle（速度降至 0）
  Idle → Held（被拾取）
  Rolling → Held（被拾取）
  ```

- **伪 3D 表现**：
  - 球有 X/Y 平面位置 + Z 轴高度
  - Z 轴高度影响阴影位置偏移
  - 碰撞检测基于 X/Y 平面，但需考虑 Z 轴（球在高处不会击中地面目标）
