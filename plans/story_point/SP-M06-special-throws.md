## 验收标准

- [ ] 全部 17 个必杀技可正常触发
- [ ] 每种必杀技有独特轨迹表现
- [ ] 必杀技伤害倍率正确
- [ ] 必杀技接球窗口差异化（普通/困难/极难）
- [ ] 分裂球幻影正确显示和消失
- [ ] 炸弹球范围伤害正确（主目标 100%，范围内 50%）
- [ ] 低 HP 时超级必杀正确触发和加强
- [ ] 必杀技不与普通投球冲突

## 技术要点

- **数据驱动架构**（应对 RISK-06）：
  ```
  SpecialThrowData (Resource):
    - trajectory_type: enum
    - damage_multiplier: float
    - catch_difficulty: enum (NORMAL/HARD/EXTREME)
    - trigger_condition: enum (DASH_THROW/JUMP_THROW)
    - visual_effect: PackedScene
    - trajectory_params: Dictionary
  ```
