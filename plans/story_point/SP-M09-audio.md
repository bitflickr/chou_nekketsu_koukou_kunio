## 验收标准

- [ ] 主菜单有背景音乐循环播放
- [ ] 比赛中有节奏感的 BGM
- [ ] 投球/接球/击中有即时音效反馈
- [ ] 必杀技发动有特殊音效
- [ ] 胜利/失败有对应音乐
- [ ] 音量整体平衡，SE 不被 BGM 淹没
- [ ] 音频管理器支持全局静音/音量调节

## 技术要点

- **音频格式建议**：
  | 类型 | 格式 | 采样率 | 说明 |
  |------|------|--------|------|
  | BGM | OGG Vorbis | 44100Hz | 循环播放，文件小 |
  | SE | WAV | 22050Hz | 低延迟即时播放 |

- **AudioManager 架构**：
  ```gdscript
  # audio_manager.gd (AutoLoad)
  var bgm_player: AudioStreamPlayer
  var se_players: Array[AudioStreamPlayer]  # SE 池（4~8个）
  
  func play_bgm(stream: AudioStream, fade_in: float = 0.5)
  func stop_bgm(fade_out: float = 0.5)
  func play_se(stream: AudioStream)
  func set_bgm_volume(db: float)
  func set_se_volume(db: float)
  ```

- **音效需求清单**：
  | 分类 | 音效 | 数量 |
  |------|------|------|
  | BGM | 主菜单/比赛×2/胜利/失败/通关 | 6 |
  | SE-球 | 投球/接球/击中/反弹/必杀技发动 | 5+ |
  | SE-角色 | 跳跃/被击中/淘汰/欢呼 | 4 |
  | SE-系统 | 菜单选择/哨声开始/哨声结束/场景切换 | 4 |
  | **合计** | | **~19** |

- **音频来源方案**：
  - AI 生成（Suno/Udio 用于 BGM）
  - sfxr/jsfxr 生成 8-bit SE
  - 开源素材库（freesound.org）
