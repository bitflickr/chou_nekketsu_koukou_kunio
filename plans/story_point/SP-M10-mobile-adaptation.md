## 技术要点

- **性能目标**（来自 GDD）：
  | 指标 | 目标 |
  |------|------|
  | 帧率 | 稳定 60fps |
  | 加载时间 | < 2 秒 |
  | 安装包 | < 50MB |
  | 最低设备 | Android 8.0+ / iOS 13+ |
  | 内存占用 | < 200MB |

- **存档数据结构**（关联 RISK-04）：
  ```gdscript
  var save_data = {
    "slot_id": int,
    "mode": String,          # "world_cup" / "vs" / "bean_ball"
    "current_match": int,    # 世界杯第几场 (0-6)
    "player_team": String,   # 玩家队伍 ID
    "timestamp": String,     # 存档时间
    "version": String        # 游戏版本（兼容性）
  }
  ```

- **性能优化策略**：
  - 像素游戏天然轻量，主要关注点：
    1. 必杀技粒子特效限制最大粒子数
    2. 使用对象池管理球和特效实例
    3. AI 决策分帧计算（不同角色不同帧更新）
    4. 减少 `_process` 中的分配
